Como reciclar de manera correcta una lata:

Para reciclar este material de manera correcta, debes seguir los siguientes pasos:

- Enjuagar la lata para eliminar los restos de residuos que esta pueda contener.

- Opcionalmente se puede aplastar la lata para que asi en el contenedor donde se inserte el material, tenga mas espacio para lograr que se puedan reciclar la mayo cantidad de latas en el espacio del contenedor.

- Si ves que la lata puede traer otro tipo de material que se pueda retirar fácilmente, quítalo.

Luego de estos pasos puedes insertar la lata en el contenedor correspondiente (color gris)

¿Sabias que?

- Las latas puedes reutilizarlas para la creación de diferentes manualidades como por ejemplo: Crear un organizador. hacer moldes para galletas, una alcancia para guardar dinero, etc.

Por esto es muy importante reciclar, ya que a pesar de una acción tan sencilla como lo puede ser desechar una lata, podemos generar muchos beneficios para nuestro planeta.
