Como reciclar de manera correcta las latas:

    para Reciclar este material de manera correcta, debes seguir los siguientes pasos :

  1. Eliminar los restos de residuos que pueda contener la lata como líquido, alimento, etc.
  2. Opcionalmente se puede aplastar la lata para que asi en el contenedor donde se inserte el material, tenga mas espacio para lograr que se puedan reciclar la mayor cantidad de latas en el espacio del contenedor.
  3. Si ves que la lata puede traer otro tipo de material que se pueda retirar fácilmente, quítalo.
  4. Luego de estos pasos puedes insertar la lata en el contenedor correspondiente (color gris).


    5. ¿Sabias que?

    Las latas puedes reutilizarlas para la creación de diferentes manualidades como por ejemplo: Crear un organizador, hacer moldes para galletas, una alcancia para guardar dinero, etc.
    Por esto es muy importante reciclar, ya que a pesar de una acción tan sencilla como lo puede ser desechar una lata, podemos generar muchos beneficios para nuestro planeta.
